This page focuses on programming strategies and techniques behind procedural analysis and generation of text-bases data. We'll explore topics ranging from evalutating text according to its statistical properties to the automates production of text with probabilistic methods to text visualization. People can learn server-side and client-side javaScript programming and develop projects that can be shared and interacted with online.
